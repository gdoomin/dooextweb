# Frontend

Next.js App Router 기반 프론트엔드입니다.

## 현재 구현

- KML 파일 선택
- `/api/convert` 호출
- 변환 결과 텍스트 표시
- 결과 요약 카드와 테이블 표시

## 환경 변수

`.env.local`

```env
NEXT_PUBLIC_API_BASE_URL=http://127.0.0.1:8000
```

## 실행

```powershell
cd C:\Users\indoo\Desktop\DH_APP\DOO_EXTRACTOR_Web\frontend
npm run dev
```
