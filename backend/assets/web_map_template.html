겹

기상

실험용 실황/준실황 오버레이입니다. 체크하면 현재 지도 범위 기준으로 외부 기상 정보를 겹쳐 보여줍니다.

형광펜을 긋습니다

완료 강조선 색상과 굵기를 선택하세요. 굵기는 다른 선 간섭을 줄이기 위해 6~10px로 제한됩니다.

거리 측정 설정

측정 결과는 버튼이 켜진 동안 계속 누적됩니다. 생성된 선/거리 라벨을 클릭하면 즉시 삭제됩니다.

강제번호 옵션

번호 라벨을 클릭해 선택한 뒤 Delete 키를 누르면 지워집니다.

이미지 저장

현재 지도 화면을 용지 크기에 맞춰 저장하거나 출력합니다.

용지

저장 크기

상단 메뉴와 패널은 제외하고 현재 지도 부분만 저장하거나 출력합니다. 출력 버튼은 브라우저의 기본 프린터 창을 엽니다.

미리보기 안에서 휠로 확대/축소하고 드래그로 이동할 수 있습니다. 기본 용지는 A4 세로입니다.

글자 입력
